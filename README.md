# Madhav Enterprises Sales Dashboard

### Overview

This is a Power BI project I put together for Madhav Enterprises to analyze their e-commerce sales. It takes raw data and turns it into an interactive dashboard to track KPIs, regional performance, and sales trends. The complete project, from the data model to the final report, is inside the **Madhav Enterprises.pbix** file.

### Tools I Used

* **Power BI:** For building the dashboard and connecting the data.
* **Power Query:** For cleaning up the raw data and getting it ready to use.
* **DAX:** For creating custom calculations like Average Order Value (AOV).

### How I Built It

1. **Data Import:** Brought in two raw CSV files (`Orders` and `Details`).
2. **Data Cleaning:** Used Power Query to fix data types, handle missing values, and group the data.
3. **Data Modeling:** Linked the two tables together using the `Order ID`.
4. **Custom Math:** Wrote DAX measures to calculate profit margins and AOV.
5. **Design:** Built out the visuals and added interactive filters.

### What's in the Dashboard?

* **KPI Cards:** Quick glances at Total Revenue, Profit, Quantity sold, and AOV.
* **Trend Charts:** A look at how profit changes month over month.
* **Category Breakdowns:** Charts showing which products and payment methods are the most popular.
* **Geographical Map:** A map showing sales across different states in India to see where the best customers are.
* **Filters:** Interactive slicers so you can easily sort the data by Quarter or State.

### Why It Matters

This project shows how data can help business owners figure out exactly what's selling, who is buying it, and how to improve their profit margins. It's a great example of taking messy data and turning it into something genuinely useful.
